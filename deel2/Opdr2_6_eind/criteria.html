Minimum vereisten

Gebruik een duidelijk onderwerp voor jouw website (0,5 punt):
Zoals eerder vernoemd gaat de website over de opdracht zelf. Het is de eindopdracht van Module 1. Daarnaast is er ook een "contact" en "About me" pagina.

Gebruik meerdere pagina's in jouw website (0,5 punt):
Ongetwijfeld is u opgevallen dat er 4 pagina's zijn; U bent immers via de menu knop hier terecht gekomen.

    Gebruik minsten 5 verschillende HTML elementen (0,5 punt):
  • main element;
  • img element;
  • div element;
  • input element;
  • form element.

Gebruik semantische HTML elementen op de juiste plaatsen & betekenis (0,5 punt):
Er is gebruik gemaakt van de header element, maar ook de main element.

Gebruik CSS in een losse stylesheet (geen inline CSS of CSS frameworks) (0,5 punt):
Natuurlijk houd ik me hier ook aan. Elke html bestand heeft een link die verwijst naar de css stylesheet.(layout.css)

Gebruik minstens op 2 plekken, padding margin of borders op jouw website (0,5 punt):
De div-tjes van elke stap in deze lijst hebben een margin van 0.5em en de unordered lists een padding van 5px.

  • Gebruik minstens 1 plaatje. (0,5 punt)
  • Gebruik minsten 2 links. (0,5 punt)
  • Gebruik 1 formulier op jouw website.(1 punt)

Gebruik MediaQueries, zodat je website er anders uit ziet op mobiel en op desktop. (0,5 punt)
Op dit moment kijkt u op een klein scherm; vergroot het scherm om het verschil te zien.

Gebruik MediaQueries, zodat je website er anders uit ziet op mobiel en op desktop. (0,5 punt)
Op dit moment kijkt u op een groot scherm; verklein het scherm om het verschil te zien.

Gebruik CSS-grid om een layout te maken en al je elementen op de pagina te positioneren (1 punt)
De id selector "container" is een grid box dat bestaat uit de Nav, Header, Main en Footer.

Gebruik flex-box om elementen te positioneren.(1 punt)
En de Menu knoppen zijn gemaakt door middel van flex-box.

Gebruik minstens 1 formulier op jouw website. (1 punt)
Klik hier om naar de contact pagina te gaan.(formulier)

Jouw website is mobile-first. (1 punt)
De basis CSS is voor een mobiele telefoon, de extra CSS voor op desktop staat in een media-query.

Overigens wordt er gekeken naar de netheid (geen "spelfoutjes") en is er gebruik gemaakt van linter. (0,5 punt)

Ook moet de code juist geformatteerd zijn. Dus spaties en enters op de juiste manier. (0,5 punt)
